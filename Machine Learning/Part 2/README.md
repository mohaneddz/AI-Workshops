# Machine Learning - Part 2

This part covers unsupervised learning and feature engineering.

## Structure

- `Templates/`
  - `Anomaly Detection.ipynb`
  - `Association Rules.ipynb`
  - `Clustering.ipynb`
  - `Dimensionality Reduction.ipynb`
  - `Feature Engineering.ipynb`
- `Exercises/`
  - each task has only `train.csv`, `test.csv`, and one exercise notebook
- `Solutions/`
  - each task has full `data.csv`, `solution.csv`, and one solved notebook
- `Challenge/`
  - realtime global network monitoring MVP with anomaly flagging

## Kaggle datasets used

- Anomaly Detection: `mlg-ulb/creditcardfraud`
- Association Rules: `heeraldedhia/groceries-dataset`
- Clustering: `vjchoudhary7/customer-segmentation-tutorial-in-python`
- Dimensionality Reduction: `uciml/iris`
