<h1 style="font-family: Arial, sans-serif; font-size: 36px; color: #4C9CEB; border-bottom: 3px solid #4C9CEB; padding-bottom: 8px;">
  AI Workshops - Practical Machine Learning Training Repo
</h1>

AI Workshops is a practical learning repository for machine learning training sessions, challenges, and guided exercises.
It includes both workshop templates and reference solutions organized by module parts.

---

## What This Repo Contains

- Part-based workshop content (`Machine Learning/Part 1`, `Part 2`)
- Challenges with datasets, requirements, and expected deliverables
- Exercises + solved notebooks for supervised and unsupervised topics
- Reusable template notebooks for rapid workshop starts

---

## Structure

```text
Machine Learning/
|-- Part 1/
|   |-- Challenge/
|   |-- Exercises/
|   |-- Solutions/
|   `-- Templates/
`-- Part 2/
    |-- Challenge/
    |-- Exercises/
    |-- Solutions/
    `-- Templates/
```

---

## Typical Workflow

1. Open the target part folder.
2. Install dependencies from that part/challenge `requirements.txt`.
3. Run notebooks in order (preprocessing -> analysis -> modeling).
4. Compare your output with solution notebooks when available.

---

## Notes

- Datasets are included for local, offline workshop delivery.
- Keep exploratory outputs out of committed source unless required by workshop instructions.
